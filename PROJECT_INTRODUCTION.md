# Line Figure Animation - 线条人物动画制作工具

Line Figure Animation 是一款基于 Web 的线条人物动画制作工具，允许用户通过简单的拖拽操作创建流畅的人物动画，并导出为 MP4 视频文件。

## 🎯 主要功能

### ✨ 快速动作
- **基础动作**：抬手、叉腰、摇头、伸直手臂等
- **舞蹈动作**：双脚开合、腿部弯曲、跑步姿势等
- **特色动作**：新增双手交叉覆盖动作（左手覆盖右手、右手覆盖左手）
- **自定义动作**：用户可以保存自己的独特动作，并在快速动作区随时调用

### 🎬 智能动作序列生成器
- **关键帧设定**：按顺序选择关键动作，系统自动生成平滑过渡
- **关键帧预览**：每个关键帧旁都有小预览画布，直观查看动作效果
- **左右布局**：动作列表与预览画布左右分栏显示，操作更便捷
- **自动换行**：动作按钮支持自动换行，适应不同屏幕尺寸

### 🎨 动画编辑
- **关节拖拽**：通过拖拽关节点精确调整人物姿势
- **帧管理**：添加、删除、复制、粘贴帧
- **一键镜像**：将当前帧的姿势进行水平翻转镜像
- **网格显示**：辅助对齐和构图

### 📁 数据管理
- **自定义动作库**：保存常用动作，支持本地存储
- **导入导出**：可导出自定义动作库，与其他用户分享
- **快捷键支持**：
  - `Ctrl+B`：打开保存自定义动作对话框
  - `Ctrl+Q`：保存自定义动作
  - `ESC`：关闭对话框

### 📤 视频导出
- **MP4 格式**：支持导出高质量 MP4 视频
- **多种导出方式**：FFmpeg 和 WebCodecs 两种导出选项
- **参数调节**：可设置拍子数、帧率等参数

## 🛠 技术特点

- **前端技术栈**：Vue 3 + Vite
- **动画引擎**：Canvas 2D 绘制
- **视频处理**：FFmpeg.wasm 和 WebCodecs API
- **响应式设计**：适配不同屏幕尺寸
- **本地存储**：利用浏览器 IndexedDB 缓存 FFmpeg 文件

## 🚀 使用场景

- **教学演示**：制作简单的人物动作演示
- **动画原型**：快速验证动画创意
- **社交媒体**：创建有趣的线条动画
- **学习工具**：了解动画制作原理

## 📋 安装与使用

```bash
# 克隆项目
git clone https://github.com/your-repo/line-figure-animation.git

# 进入项目目录
cd line-figure-animation

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```
